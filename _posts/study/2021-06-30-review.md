# 주석 (Comment)
- 프로그래머가 하고 싶은 말을 메모해두는 것
- 주석은 프로그램 실행에 영향을 미치지 않는다

// 한줄 주석  
/**/ 여러줄 주석  

; (세미콜론) : 한 명령어가 끝났음을 알린다.

""사이에 적는 것 : 문자열 (String), 문자 여러개  
숫자 : 정수 (int, Integer)  
소수 : 실수 (Double, Float)  
''사이에 적는 것 : 문자 (Character), 문자 단 하나
